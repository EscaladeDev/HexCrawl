# Hex Crawl

A lightweight browser-based hex exploration tool for tabletop prep, solo/worldbuilding play, or real time exploration at the table.

Hex Crawl lets you reveal adjacent hexes, generate points of interest, tune discovery odds, and keep a simple exploratory map moving.

## What it does

- Explore adjacent hexes from a starting position
- Reveal discovered hexes as you travel
- Toggle between **d20 table** mode and **weighted** mode
- Enable or disable **fog of war**
- Enable or disable **patrols**, with configurable spawn and move chances
- Adjust visibility state with **disadvantage / normal / advantage**
- Use an optional **seed** for repeatable generation
- Include extra generated detail like:
  - clarity
  - secondary results
  - where-in-hex prompts
- Edit option weights live
- Save and load configuration as JSON
